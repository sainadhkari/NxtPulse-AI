## PRODUCT OVERVIEW

Build a full-stack AI-powered SaaS dashboard web application.

Purpose:
This product helps managers, POCs, and SDI trainees in technical training programs (like CCBP/NxtWave) monitor learning quality, detect struggling trainees early, reduce manager workload, and improve demo performance.

This is NOT a normal website.

This must feel like a premium AI SaaS product demo for hackathons, investors, and judges.

Think:
Palantir + Notion + modern AI dashboards.

Use:

* Dark futuristic theme
* Glassmorphism
* Blue/cyan neon accents
* Animated charts
* Premium cards
* Responsive design

Tech Stack:
Frontend:

* React
* TypeScript
* Tailwind CSS

Backend:

* Supabase / Firebase / Node backend

Optional:

* OpenAI / Gemini / Claude API for AI features

---

## PROBLEM STATEMENT

Current problems in SDI training systems:

1. One manager handles 50–60+ trainees
2. POCs manually track progress in sheets
3. Weak trainees remain hidden
4. AI dependency is increasing
5. Demo quality is inconsistent
6. Managers get delayed intervention signals
7. Too much manual follow-up work

Need:
An AI system that detects struggling trainees early and automates monitoring.

---

## CORE AI MODULES

1. LearnGuard AI
   Purpose:
   Evaluate whether trainee actually understands concepts.

Checks:

* Understanding
* Confidence
* AI Dependency
* Readiness

Example:
Trainee says:
"I completed React Task"

AI asks:

* Why use useState?
* Why not props?
* Explain re-rendering.

AI generates scores.

---

2. Demo Intelligence
   Purpose:
   Evaluate demo quality.

Checks:

* Technical strength
* Communication
* Confidence
* Teaching readiness

Output:
AI feedback on demo quality.

---

3. Silent Detector AI
   Purpose:
   Detect hidden struggling trainees.

Signals:

* Attendance drop
* Demo quality drop
* AI dependency increase
* Reduced engagement
* Poor standups

Output:
Risk level + intervention suggestion.

---

4. Understudy AI
   Purpose:
   AI simulates POC / manager routine work.

Can handle:

* reminders
* follow-ups
* escalation
* reporting

This is the WOW feature.
